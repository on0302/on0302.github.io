MEMORY

メモリーとは、アプリケーションやシステムの動作に必要なファイルを一時的にほ保存するパーツです。メモリーの容量が多くなればなるほど、１度にたくさんのファイルを扱うことが可能になり、結果として動作が高速化します。

選び方のポイント

インターフェイス(形式)

メモリーでは、「DIMM DDR4」や「S.O.DIMM DDR3」といった分類をされます。このうち、DIMMやS.O.DIMMにあたる部分が、インターフェイス(形式)です。DIMMはデスクトップ用、S.O.DIMMはノートパソコン用を意味しています。また、DDR以降の部分が規格です。

DIMM(デスクトップ用)

「Dual In-line Memory Module」の略称で、主にデスクトップ用で使用されます。形状が細長く、メモリスロットに合わせるり込みがあるのが特長です。

S.O.DIMM(ノートパソコン用)

「Small Outline Dual In-line Memory Module」の略称で、大きさはDIMMの約半分です。主にノートパソコン用に使用されますが、デスクトップ一体型でも使用されることがあります。

規格

DDR4 SDRAM

DDR3の2倍の転送速度と、消費電力が約20%向上した次世代メモリーです。まだ、一部のマザーボードのみ対応しているが今後主流になると思われるメモリー。

DDR3 SDRAM

DDR2の後継規格で2007年に登場。さらなる高速化と消費電力低減を実現。現在の主流規格です。

DDR2 SDRAM

2005年〜2009年頃に販売されていたパソコンに使われていた規格です。

周波数

メモリーは、メモリチップの規格、またはメモリモジュールの規格が決まっており、それぞれから動作クロック周波数、データ転送速度がわかります。
