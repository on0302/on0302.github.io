OS

OSは、コンピューターを動かす上で基本となるソフトで、OSを基本ソフトウェアといいます。Operating System(オペレーティングシステム)の略で、コンピューター全体を管理、制御し、人が使えるようにする役割があります。

接続インターフェイス

HDDの接続インターフェイスには大きく2つの種類があります。IDEとSATAがあります。 現在ではSATAが主流になっていますが、一世代前のパソコンなどにはIDEのHDDが載っていることもあります。SATAは性能も価格もIDEより優れているので、選ぶ際はSATAのものを購入しましょう。 また、SATAにもいくつかの規格があります。それぞれの規格によって転送速度なども変わるので、最新のものを選ぶようにしましょう。

規格 転送速度
SATA3.0 6Gbps
SATA2.0 3Gbps
SATA1.0 1.5Gbps
