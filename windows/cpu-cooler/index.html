CPU COOLER

CPUは常に熱を発しています。また重たい処理や長時間使用していると温度が上昇してきます。高温状態のままにしておくとCPUの寿命が低下するほか、破損につながります。そこでCPUクーラーでCPUを冷却し、熱による暴走を防いで、動作を安定させているのです。CPUクーラーには主に2種類の方式があります。

CPUクーラーを選ぶ上でのポイント

空冷式

CPUクーラーで主流の方式です。ヒートシンクと呼ばれる金属部分に冷却ファンを使用して風を当て熱を奪う方法です。最近では静音性の高い製品も登場しています。

水冷式

空冷式よりも冷却効果の高い方式です。CPUから冷却液に熱を移し、ラジエーターで冷却液を循環させ冷却ファンで冷やし排熱する仕組みです。冷却効果が高いですが、冷却液の補充などのメンテナンスが必要です。最近ではメンテナンスフリーの簡易水冷式も登場しています。

選ぶ方のポイント

CPUソケット

装着するCPUソケットの規格とあったものを選ぶ必要があります。CPUクーラーには数種類の規格に対応したアタッチメントが付属していますが、取り付けたいマザーボードのソケット形状にあってるかどうかを事前に必ず確認しましょう。

サイズ

ソケット形状があっていても、使いたいPCケースに収まるサイズでなければCPUクーラーを取り付けることができません。サイズのチェックも必ずしましょう。

サイドフローとトップフロー

空冷式CPUクーラーはサイドフロー型とトップフロー型の2種類に分けられます。トップフロー型はCPUだけでなくマザーボード全体やメモリも冷却できるのが特徴です。サイズも豊富で省スペースPCやスリムケースにも対応しています。しかし、CPU冷却後の空気をちゃんと排気する必要があります。いっぽう、サイドフロー型はCPU冷却後の空気を背面ファンで排気する事ができます。風の流れが直線的なので、冷却後の空気が溜まりにくくCPUを冷却できます。CPU以外のパーツの冷却が弱くなってしまうというデメリットも有ります。こういった特徴を踏まえてタイプを選びましょう。
