CPU

CPUは、パソコンでもっとも重要になるパーツで、演算処理を行う装置でパソコンの頭脳となる部分になります。
CPU自体は小さなものですが、パソコンの性能を大きく左右します。
CPUは、インテルとAMDの二社が占めています。

CPUを選ぶ上でのポイント

 ポイント 
説明
価格 性能の高いCPUになるにつれて価格も高くなります。価格がおなじくらいであれば、性能も似たものになります。
ソケット形状 CPUはマザーボード上にあるソケットに装着しますが、ソケットはCPUのメーカーやシリーズによって形状が異なります。そのため、CPUを選ぶ際にはマザーボート側のソケットタイプにあった物を選びましょう。
TDP CPUの最大発熱量を表すもので、CPUクーラーを選ぶ際の基準になります。この値が高いCPUは、大きいCPUクーラーなどでないと動作が安定しなくなることもあります。
コア 演算処理を行う部分で、デュアルコアの場合にはシングルコアCPUの２つ分の処理を並行して行うことができます。コア数が多いほど、複数の処理を手分けしてすばやく行うことができます。
スレッド 一度に処理できる命令の数のこと。コア数とスレッド数はおなじになりますが、インテルのCPUは、仮想的に倍のスレッドで処理することができるハイパースレッディングという機能を備えているものが多いです。
動作周波数 クロックとも呼ばれます。コア数などが同一のCPUであれば、動作周波数の値が大きいほど、処理速度は高速になり、TDPも大きくなります。
キャッシュメモリー CPU内部に設けたれた記憶装置。メインメモリーへのアクセスを減らして高速化してくれます。
GPU性能 内蔵グラフィックスと呼ばれます。3Dゲームなどには重要担ってきます。最近のCPUは内蔵グラフィックスも性能が上がっているため、通常の使用では、あまり変わりはないでしょう。intelのCPUはCPU性能、AMDのCPUはGPU性能が高い傾向があります。
